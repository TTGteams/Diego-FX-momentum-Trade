## Docker
### Build Service
```bash
docker-compose build
```
### Run Service
```bash
docker-compose up
```

## Local
### Virtual Environment
```bash
python -m venv ./nautilus-env
```

### Activate
```bash
nautilus-env\Scripts\Activate.ps1
```

### Install Deps
```bash
python -m pip install -r requirements.txt
```

### Run server
```
python server.py
```
